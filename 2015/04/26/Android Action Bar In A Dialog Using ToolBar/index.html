Android Action Bar In A Dialog Using ToolBar

  • Android
  • UI
  • 翻译
  • 原文链接 http://www.truiton.com/2015/04/android-action-bar-dialog-using-toolbar
  • 原文作者 Mohit Gupt


Android Toolbar 是 AppCompat v21包中一个新的UI组件。Toolbar 跟ActionBar 一样，唯一的区别是，Toolbar可以放置在屏幕的任意位置。Toolbar还可以用来替换你的Activity中原本的ActionBar。这个特性带给应用程序许多新的可能性。例如你可以在屏幕上放置多个操作栏，还可以放置有具体尺寸的无需覆盖整个屏幕宽度的操作栏/工具栏。Toolbar还解决了我在开发过程中面临的一个重要问题，即在一个dialog fragment中放置一个Android ActionBar。现在有了Toolbar，这个问题很容易就解决了。

Toolbar支持logo图片,标题和副标题。它还支持导航按钮,自定义视图,更重要的是它可以支持操作菜单。现在的问题是如何使用Toolbar在dialog上显示操作栏。
